## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

Key Concepts:

1. Hooks
2. Custom hooks
3. Debounce
4. Search
5. Table
6. Pagination
7. Api Fetching
8. Array Filtering and Mapping
9. Portal
10. Higher Order 
11. Routing
12. Navigation
13. useCallback
